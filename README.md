# 北京积分落户计算器

一个精美的H5工具，帮助用户计算北京积分落户分数，预测落户时间，并提供个性化的加分建议。

## 功能特点

- 📊 **智能计算积分**：基于北京市积分落户政策，全面评估9大积分项
- 📈 **趋势预测分析**：可视化展示分数增长趋势，预测达标年份
- 💡 **个性化建议**：根据用户情况，提供可操作的加分方案
- 📱 **移动端优化**：专为手机浏览器设计，操作流畅美观
- 🔄 **数据持久化**：自动保存填写数据，避免重复输入
- 📲 **分享功能**：支持链接分享和二维码扫码访问

## 技术栈

- **框架**：React 18 + Vite 4
- **UI 库**：Ant Design Mobile 5.x
- **图表**：ECharts 5.x
- **日期处理**：Day.js
- **二维码**：qrcode.react

## 积分规则说明

本工具基于北京市积分落户政策实现，包含以下积分项：

### 1. 学历积分（一次性）
- 大学专科：10.5分
- 大学本科：15分
- 硕士研究生：26分
- 博士研究生：37分

### 2. 社保年限
- 每缴纳满1年：3分

### 3. 年龄
- 45岁以下：不扣分
- 超过45岁：每增加1岁扣4分

### 4. 居住
- 自有住房：每满1年积1分
- 租房：每满1年积0.5分

### 5. 职住匹配
- 就业地和居住地都在城六区外：每满1年积2分，最高12分

### 6. 纳税
- 近3年连续缴纳个税10万+：6分

### 7. 创新创业
- 发明专利：每项9分
- 实用新型专利：每项3分

### 8. 荣誉表彰
- 省部级以上荣誉：20分
- 见义勇为等：20分

### 9. 守法记录
- 违法记录：每次扣30分

## 部署说明

### 1. 修改分享链接

编辑 `src/components/SharePanel.jsx`，将占位URL替换为实际域名：

```javascript
const shareUrl = 'https://your-domain.com';  // 替换为实际URL
```

### 2. 静态托管部署

构建后的 `dist/` 目录可以部署到任何静态托管服务：

- Vercel
- Netlify
- GitHub Pages
- 阿里云 OSS
- 腾讯云 COS

### 3. 服务器部署

如果部署到自己的服务器，需要配置 Nginx/Apache 支持 SPA 路由。

Nginx 配置示例：

```nginx
server {
    listen 80;
    server_name your-domain.com;
    root /path/to/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

## 数据隐私

- 所有计算均在浏览器本地完成，不会上传到服务器
- 用户数据保存在浏览器的 localStorage 中
- 可随时清除浏览器数据来删除个人信息
